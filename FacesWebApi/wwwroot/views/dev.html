Open Web Api for developers

User controller '/api/user/'

  • Get({id: int})
    Optional parameters(from query):
    • WithRole: bool
    • WithRequests: bool
    • FromRequest: int?
    • RequestsCount: int?
    • WithRequestImages: bool
    • WithRequestResponses: bool
    • WithResponseImages: bool
    Return a user object only without password, and with the appropriate properties.
    Return 404 if user with same id not exist.
  • Post()
    Necessary parameters(from body):
    • Nickname: string
    • Email: string
    • Password: string
    Register user with default user role.
    Return 409 if user with same email already exist.
    Return 201 with account url and user id, if user was created.
  • Put({id: int})
    Necessary parameters(from body):
    • Nickname: string
    • Email: string
    Optional parameters(from body):
    • Password: string
    Change user data.
    Return 409 if user with same email already exist.
    Return 400 if non-authorized user.
    Return 404 if user with same id not exist.
    Return 202 if data succeesful changed.
  • Delete({id: int})
    Delete user by id.
    Return 400 if non-authorized user.
    Return 404 if user with same id not exist.
    Return 200 if user data succeesful delete.
