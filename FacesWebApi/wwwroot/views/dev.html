Open Web Api for developers

User controller '/api/user/'

  • Get({id: int})
    Optional parameters(from query):
    • WithRole: bool
    • WithRequests: bool
    • FromRequest: int?
    • RequestsCount: int?
    • WithRequestImages: bool
    • WithRequestResponses: bool
    • WithResponseImages: bool
    Return a user object only without password, and with the appropriate properties.
    Return 404 if user with same id not exist.
  • Post()
    Necessary parameters(from body):
    • Nickname: string
    • Email: string
    • Password: string
    Register user with default user role.
    Return 409 if user with same email already exist.
    Return 201 with account url and user id, if user was created.
  • Put({id: int})
    Necessary parameters(from body):
    • Nickname: string
    • Email: string
    Optional parameters(from body):
    • Password: string
    Change user data.
    Return 409 if user with same email already exist.
    Return 400 if non-authorized user.
    Return 404 if user with same id not exist.
    Return 202 if data succeesful changed.
  • Delete({id: int})
    Delete user by id.
    Return 400 if non-authorized user.
    Return 404 if user with same id not exist.
    Return 200 if user data succeesful delete.

Request controller '/api/request/'

  • Get({id: int})
    Return a request model with response.
    Return 404 if request with same id not exist.     Response model
    • Request
      • RequestId: int
      • RequestType: string
      • Images: array of objects {ImageSrc: string}(local img url)
    • Response
      • Images: array of objects {ImageSrc: string}(local img url)
    • User
      • UserId: int
      • Username: string
  • Post()
    Necessary parameters(from form):
    • RequestType: string
    • FromImage: image file
    Optional parameters(from form):
    • ToImage: image file(if requestType == SwapRequest)
    Request types:
    • SwapRequest: swap faces from fromImage to ToImage.
    • DetectRequest: detect all faces on image.
    • CutRequest: Cut all faces from image.
    Create request and response of face request.
  • Delete({id: int})
    Delete request and response by id.
    Return 400 if non-authorized user.
    Return 404 if not found request with same id.
    Return 200 if data succeesful delete.

News controller '/api/news/'

  • Get()
    Return a news list. Necessary parameters(from form):
    • From: int
    • Count: int
    • WithBody: bool
  • Get({id: int})
    Return a news.
    Return 404 if news with same id not found.
